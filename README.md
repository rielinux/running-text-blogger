# Running Text Blogger

Widget running text sederhana untuk Blogger.

## Fitur

- Warna teks dapat diubah
- Warna background dapat diubah
- Border dapat diubah
- Kecepatan dapat diatur
- Responsive

## Cara Menggunakan

Salin kode dari file `index.html` ke template Blogger Anda.
